== Description ==

Saul is a clean and creative Jax Lite child theme based on Bootstrap framework, with a different layout for some elements like the header, post details, buttons and new Google Fonts. Saul is optimized to work with Contact Form 7 plugins.

Created by ThemeinProgress, http://www.themeinprogress.com
